This page is intended as a partial tutorial to make your own website. I'll outline the steps I took to make this page,
and add links to additional information. This is a work in progress and wil evolve as I have time to write it.

Building the site
----
There are multiple different solutions to build your own webpage. For fast load times, easy content management, and general
ease of use I built mine using the [Hugo](https://gohugo.io) static site generator.

Hosting
----
As the easiest solution, I use GitHub Pages to deliver my site.

Domain name
-----
GitHub Pages allows mapping a custom domain name to your hosted site. I'm using [MythicBeasts](https://www.mythic-beasts.com)
for mine, although a myriad are available.
